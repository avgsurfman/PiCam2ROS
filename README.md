# PiCam2ROS

Example rclpy Nodes for transferring Images through ROS2 from PiCamera2.

## Installation

```
colcon build
```

## Dependencies

This example uses and requires [cv_bridge](https://github.com/ros-perception/vision_opencv/tree/rolling/cv_bridge).
